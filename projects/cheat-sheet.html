Dev Bootcamp Cheat Sheet

RUBY SYNTAX

If/Else statements:

Basic syntax:


        if 1 < 2
          print "yup"
        elsif
          print "ok"
        else
          print "war"
        end

Can refactor to:

expression if bool

Ternary conditional:

boolean ? Do this if true: Do this if false

puts var1 == var2 ? "ok" : "nope"
Unless statemnts:

Basic syntax:


        unless bool
          print "ok"
        else
          print "cool"
        end

Can refactor to:

expression unless bool
Iterators

While loop:


        counter = 1
        while counter < 11
          puts counter
          counter += 1
        end

Until loop:


        counter = 1
        until counter == 11
          puts counter
          counter += 1
        end

For iterator: (hint '..' is inclusive)


        for num in 1..10
          puts num
        end

Fixed iterator:

10.times{|n| puts n}

Array iterator:

array.each{|x| puts x}

Others:

.upto(num) num.next
Hash and Arrays

Arrays:


        my_array = [1,2,3]
        my_array.each{ |x| puts x }

Hashes:


        my_hash = {
          "Key1" => "value1",
          "Key2" => value2
        }

or


        my_hash = Hash.new
        my_hash["name"] = "Menuka"

Iterating through hash with key-value pair:

my_hash.each{|key,value| puts #{key}: #{value}}

Can also use symbols for hashes. Some other useful methods include:

.each_key .each_value .delete(value)
Methods:

Basic declaration:


        def method_name(arguments)
          code body
        end

In methods we don't need return statements and can set default parameter values if not included in the method call.

Quick tips:

Input:

input = gets.chomp numInput = Integer(gets.chomp)

Output variables

puts "Number was #{var}"

Split words to array:

wordArray = text.split

String tricks:

string.capitalize! string.upcase! string.downcase!

Quick replace char:


        if user_input.include? "s"
          user_input.gsub!(/s/ , "th")
        end

Ruby types tricks:

.to_s .is_a?

Combined comparison operator: (Returns 1,0,-1 depending on left compared to right)

num.sort!{|first_num, last_num| second_num <=> first_num}

Check if method/object/symbol responds to certain method:

my_array.respond_to?(:push)
